# Gree Developer Platform

## Project setup
```
yarn install
```

### 启动客户端
```
yarn serve
```

### 启动服务器
```
yarn start
```

### 启动服务器与客户端
```
yarn dev
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn run lint
```
