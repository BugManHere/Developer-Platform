# Gree Developer Platform

## Project setup
```
yarn install
```

### 启动客户端
```
yarn start
```

### 启动服务器，热更新
```
yarn serve
```

### 启动服务器，非热更新
```
yarn serve:const
```

### 启动服务器与客户端
```
yarn go
```

### 启动服务器与客户端，客户端本地调试模式，需要本地有数据库
```
yarn local
```

### 启动服务器与客户端，服务器本地调试模式，需要本地有数据库
```
yarn test
```

### Compiles and minifies for production
```
yarn build
```
